# Sign Language Detection (A–Z)

This project detects **English Alphabet Sign Language (A–Z)** using **Mediapipe**, **TensorFlow**, and **OpenCV**.  
It uses your webcam to capture hand landmarks and predict the corresponding letter in real-time.

---

##  Requirements

To run this project, you will need the following dependencies:

- Python 3.10 or higher  
- Mediapipe library  
- OpenCV library  
- TensorFlow library  
- Numpy library  
- Scikit-learn library  
- Matplotlib (optional for visualization)

---

##  Installation

Open your terminal or PowerShell and run these commands:

```bash
pip install opencv-python==4.11.0
pip install mediapipe==0.10.9
pip install tensorflow==2.11.0
pip install numpy==1.23.5
pip install scikit-learn==1.3.2
pip install matplotlib==3.7.2
